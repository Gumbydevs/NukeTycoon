⚠️ Danger Zone

These actions are destructive and irreversible. Confirm each before proceeding.

🔧 Maintenance
🗑️ Purge Auth Codes

Deletes all used or expired one-time login codes from auth_codes. Safe to run any time.

📜 Wipe Old Run History

Deletes all ended runs and their associated per-run data — buildings, fallout, sabotage, enrollments. The active run is untouched.

🏃 Current Run
🔄 Wipe Run Players

Removes all buy-in records for the current run. Forces every connected player back to the buy-in lobby.

🧹 Wipe Player State

Deletes all run_player_state rows for the current run — uranium, income, daily stats, scores.

🏗️ Wipe Buildings

Hard-deletes all building rows for the current run and broadcasts a grid reset to all connected clients.

☢️ Wipe Fallout Zones

Removes all active fallout/radiation zone rows for the current run.

💣 Wipe Sabotage Log

Clears all sabotage_events rows for the current run. Does not undo effects — records only.

👤 Player Accounts
💰 Reset All Balances

Sets every player's token_balance to 50,000. Does not touch any other state.

✏️ Set Player Balance

Overwrite a single player's token balance. Enter their email or username.

🗑️ Wipe ALL Players

Deletes every player account plus all runs, buildings, auth codes, and per-run data. Scorched-earth option.

☢️ Nuclear
☢️ FULL DB RESET

Wipes every run, all per-run data (buildings, fallout, sabotage, enrollments, state), resets all balances to 50,000, then starts a fresh run. Player accounts are preserved. Cannot be undone.
