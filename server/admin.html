⚠️ Danger Zone

These actions are destructive and irreversible. Confirm each before proceeding.

🔄 Wipe Run Players

Removes all buy-in records for the current run. Every connected player will be forced back into the buy-in lobby on their next action.

💰 Reset All Balances

Resets every player's token balance to 50,000. Does not touch any other state.

🧹 Wipe Player State

Deletes all run_player_state rows for the current run (uranium, income, daily stats etc.).

☢️ FULL DB RESET

Ends current run, deletes all run_players + run_player_state, deactivates all buildings, resets all player balances to 50,000, then starts a fresh run. Cannot be undone.
