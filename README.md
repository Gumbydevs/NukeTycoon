# Nuclear Tycoon

A competitive multiplayer tycoon game where players build nuclear infrastructure and sabotage competitors for real money.

## Project Structure

```
nuclear-tycoon/
├── index.html          # Main HTML file
├── styles/
│   └── main.css        # All styling
├── js/
│   └── game.js         # Game logic and mechanics
└── README.md           # This file
```
